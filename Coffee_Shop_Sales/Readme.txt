Coffee Shop Sales Dashboard Project
-----------------------------------
**Password: COFFEEdash**

This project is a comprehensive dashboard for analyzing coffee shop sales data. 
It provides interactive insights and visualizations to help manage sales, track trends, 
and make data-driven decisions.

Folder: Coffee_Shop_Sales
Files included:
- Coffee Dashboard.xlsx
- Coffee Shop Sales Dataset.xlsx
- CoffeeDashboard PNG.png

Key Features:
- Dashboard includes multiple sheets with sales analysis.
- Some sheets are protected; for example, the "Dashboard Names" sheet is password-protected. 
- Interactive visualizations using slicers, pivot tables, and charts.
- Data cleaned and structured using PowerPivot and Excel tools.
- Provides clear summaries of monthly and product-level sales.
- Easy-to-use interface for managers to explore sales data.

Purpose:
The dashboard is designed to help coffee shop managers and stakeholders 
optimize inventory, pricing, and promotional strategies using actionable insights.
